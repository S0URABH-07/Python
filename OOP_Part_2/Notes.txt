    1. Encapsulation
            What is Encapsulation?
                Encapsulation means bundling data (attributes) and methods (functions) into a single unit (class) and controlling direct access to the data.

                Real-Life Example
                    Think of an ATM.
                        You can:
                            Deposit money
                            Withdraw money
                            Check balance
                        But you cannot directly change your bank balance by opening the ATM.

                                The balance is protected.

            Private Variables:- 
                private variable can be access only in class
                    self.__balance
                            Two underscores (__) make the variable private (name-mangled). It should only be accessed through class methods.

            Protected Variable:-
                Protected variable can be access only through derive class
                    self._name

            public variable :-
                Public variable can be access whenever you want      


    2. Inheritance
            What is Inheritance?
                Inheritance allows one class to reuse the properties and methods of another class.    
                    Animal
                        │
                        ├── Dog
                        ├── Cat
                        └── Lion
                
                Types:-
                    Multilevel:-
                            Animal
                               │
                             Mammal
                               │
                             Dog

                    Multiple:-
                        Father      Mother
                             \      /
                              Child

                    Hierarchical Inheritance:-
                                    [ Parent Class ]
                                      /        \
                                     /          \
                              [ Child 1 ]    [ Child 2 ]

    3. Polymorphism
            What is Polymorphism?
                One interface, many forms.
                The same method name can behave differently depending on the object.

            Method overriding-> Diffrent class but function name is same this is called overriding


    4. Abstraction
            What is Abstraction?
                Abstraction means showing only essential features while hiding implementation details.
                            Real-Life Example
                                    When you drive a car:
                                                You use:
                                                    Steering
                                                    Brake
                                                    Accelerator
                                                You don't need to know how the engine works internally.


                        Static Method:
                            A static method does not use self or cls.
                                                class Calculator:
                                                    @staticmethod
                                                    def add(a, b):
                                                        return a + b

                        Class Method:-
                            Uses cls instead of self.
                                                class Student:
                                                    school = "ABC"
                                                    @classmethod
                                                    def show_school(cls):
                                                        print(cls.school)

Important :-class variable or Instance variable
                class variable make under the class 
                        or
                Instance variable make under the function

                        Example :-
                            class Student:
                                school = "ABC"   # Class variable
                                
                                def __init__(self, name):
                                    self.name = name   # Instance variable
